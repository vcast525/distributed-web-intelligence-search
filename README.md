# Project #9 — Distributed Web Intelligence Search Engine

A distributed backend application that collects, processes, stores, indexes, and searches web-based intelligence from approved external sources.

This project demonstrates FastAPI, Redis, Celery, MongoDB, Elasticsearch, Docker, asynchronous processing, distributed workers, search indexing, and API-driven architecture.
